Download das ocorrências da tartaruga-marinha Eretmochelys imbricata.

Vamos fazer o download destas bases de dados:

  • gbif: Global Biodiversity Information Facility (GBIF)
  • vertnet: VertNet - Dados de vertebrados de coleções científicas

24.4.5 Partição dos dados

Vamos preparar agora o partionamento dos dados em treino (ajuste) e teste (avaliação) dos modelos.

### data partitioning ----

#### pseudoabsence ----
occ_sp_filt_psa <- rbind(occ_sp_filt[, 2:4], psa)
n_part_while <- 5
min_occ <- ifelse(nrow(occ_sp_filt) >= 30, 10, 5)

occ_sp_filt_psa_part <- NA
while(any(is.na(occ_sp_filt_psa_part))){
    occ_sp_filt_psa_part <- flexsdm::part_sblock(
        data = occ_sp_filt_psa,
        env_layer = var_ca_sel,
        pr_ab = "pr_ab",
        x = "x",
        y = "y",
        n_part = n_part_while,
        min_occ = min_occ, 
        prop = .5)
    n_part_while <- n_part_while - 1
}

occ_sp_filt_psa_part_grid_ca <- flexsdm::get_block(env_layer = var_ca_sel, 
                                                   best_grid = occ_sp_filt_psa_part$grid)

n_part_color <- cols4all::c4a("magma", n = max(occ_sp_filt_psa_part$part$.part))
n_part_color_occs <- cols4all::c4a("set1", n = max(occ_sp_filt_psa_part$part$.part))

tm_shape(brazil_buffer) +
    tm_borders() +
    tm_shape(brazil) +
    tm_fill() +
    tm_shape(occ_sp_filt_psa_part_grid_ca) +
    tm_raster(col.legend = tm_legend(show = FALSE),
              col.scale = tm_scale_categorical(values = n_part_color)) +
    tm_shape(ca) +
    tm_borders(col = "red") +
    tm_shape(terra::vect(occ_sp_filt_psa_part$part, geom = c("x", "y"), crs = "EPSG:4326")) +
    tm_dots(fill = ".part",
            shape = "pr_ab",

24.4.7 Ajuste dos modelos

Agora sim, podemos ajustar os modelos. Vamos usar quatro algoritmos:

  • GAM: estatístico com presença e pseudoausência
  • GAU: estatístico com presença e pontos de fundo
  • GBM: machine learning com ajuste de hiperparâmetros com presença e pseudoausência
  • MAX: machine learning com ajuste de hiperparâmetros com presença e pontos de fundo
### models ----
models_fitted <- list()

#### gam ----
fit_gam <- flexsdm::fit_gam(
    data = occ_sp_filt_psa_part_data,
    response = "pr_ab",
    predictors = var_ca_sel_names,
